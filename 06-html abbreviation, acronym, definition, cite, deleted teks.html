Contoh Penggunaan Tag HTML

Dulu ketika SMA saya dan teman-teman saya sering latihan bermacam macam kegitan, seperti kaligraphi, hadroh, dll.

Saat ini aku sedang belajar basic HTML.

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat struktur dan kerangka dasar halaman website. HTML menggunakan tag dan elemen untuk mengatur konten seperti teks, gambar, video, dan tautan di browser. Ini adalah fondasi utama pengembangan web dan umumnya bekerja bersama CSS (desain) dan JavaScript (interaktivitas).

Tag yang di gunakan

Untuk abbreviation menggunakan tag <abbr;>

Untuk acronym menggunakan tag <acronym;>

Untuk definition menggunakan tag <dfn;>

Untuk cite menggunakan tag <cite;>

Untuk deleted text menggunakan tag <del;>

Untuk inserted text menggunakan tag <ins;>
